# はてなブログ Scripts

## 使い方

`管理ページ > 設定 > 詳細設定 > <head>要素にメタデータを追加`に以下を貼り付けてください。

```html
<script src="https://cateiru.github.io/hatena-blog-scripts/bin/scripts.min.js" defer></script>
```
